## üö¶ Getting Started

1. **Clone the repo:**
   ```sh
   git clone https://github.com/NishantRaj278/Portfolio.git
   cd nishant-portfolio
   ```
2. **Install dependencies:**
   ```sh
   npm install
   ```
3. **Run the development server:**
   ```sh
   npm run dev
   ```
4. **Build for production:**
   ```sh
   npm run build
   ```
